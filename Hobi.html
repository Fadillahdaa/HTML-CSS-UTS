INI ADALAH BEBERAPA HOBI SAYA

Ada beberapa hobi yang saya gemari, yaitu memasak, mendengarkan musik, dan membaca. Saya menyukai hobi saya ini, karena itu membuat saya senang. Saya tidak melakukan hobi saya sekaligus, tetapi jika saya merasa bosan dengan hobi saya yang 1 maka saya akan melakukan yang lainnya. Akan tetapi saya lebih sering melakukan hobi mendengarkan musik karena saya mendengarkannya setiap hari. Dan untuk hobi saya yang memasak, saya akan melkukannya apabila senggang. Dan untuk membaca sendiri saya melakukannya abapila mood.
