Tentang Saya

Halo, Perkenalkan Nama Saya Fadillah Dwi Anggraini, biasa dipanggil Fadillah. Saya sebagai mahasiswa aktif semester 2 di Politeknik Negeri Madiun dengan Prodi D-III Teknologi Informasi. Saya mengambil jurusan ini karena permintaan orang tua saya. Sebelumnya saya bersekolah di SMK Negeri 2 Madiun dengan Jurusan OTKP atau Otomatisasi Tata Kelola Perkantoran. Saya merupakan anak ke-2 dari 3 bersaudara. Hobi saya adalah mendengarkan musik, memasak, dan membaca.

Saya lahir di Kabupaten Pasuruan dan ketika berumur 3 tahun saya pindah ke Kota Madiun. Ketika saya sekolah dasar, saya bersekolah di TK Siwi Peni, lalu waktu sd, saya bersekolah di SDN Oro-Oro Ombo. Dan ketika SMP saya menempuh pendidikan di SMP NEgeri 8 Kota Madiun. Waktu SMA saya menempuh pendidikan di SMK Negeri 2 Madiun.
